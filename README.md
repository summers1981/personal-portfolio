# Colin Summers — Personal Portfolio

Personal portfolio website for Colin Summers, Design Architect for AI/LLMs.

## Pages

- **Home** (`index.html`) — Hero, pillars, featured work
- **Work** (`work.html`) — Filterable project portfolio
- **About** (`about.html`) — Bio, career journey, skills
- **Contact** (`contact.html`) — Contact form and social links

## Stack

Pure HTML, CSS, and vanilla JavaScript. No build step required.

## Deploy

This site is ready to deploy to GitHub Pages, Netlify, or any static host.

To enable GitHub Pages: Settings → Pages → Deploy from branch → `main` / `/ (root)`.
